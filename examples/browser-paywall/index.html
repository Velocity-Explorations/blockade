L402 paywall widget demo (v2: staked-credential cost curve). Proxy on :8080 (make up && make setup).

Session Credential

Status: Not enrolled

The widget stores your enrollment credential in sessionStorage. After the anonymous cap (5 requests), you will be prompted to enroll. Subsequent requests use the held credential with escalating prices.

Mode 1: Declarative

One attribute, one script tag, no application code. Click the link to trigger the paywall modal. Repeat to see the cost curve escalate.

<a href="http://localhost:8080/get" data-l402>Download Protected Resource</a> <script src="l402-paywall.js"></script>
Download Protected Resource

Mode 2: Programmatic

Drop-in fetch replacement with the same call signature. Each call shows the next price on the curve.

const res = await l402Fetch('http://localhost:8080/get'); const data = await res.json();
Response body

Mode 3: Global Monkey-Patch

Opt-in override of window.fetch. Every fetch that hits a 402 endpoint triggers the paywall automatically.

L402.installGlobal(); // now plain fetch() handles 402s const res = await fetch('http://localhost:8080/get');
